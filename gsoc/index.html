The Proposal

I did Aperture Photometry with a globular cluster in the winter and had trouble detecting and fitting for sources at the center of the field where you encounter a lot of overlapping sources. I was not planning to invest in coding another summer but a friend linked me this idea to implement PSF fitting for overlapped sources in Astropy photutils. I was already familiar with photutils having used it in the aforementioned project. I decided to take up this task and start working towards it.

I could see a tough path as a I started very late and looking at the competiton, many suggested changing the topic, but very frankly I did not want to do GSoC if not for this project being offered so here's my proposal
