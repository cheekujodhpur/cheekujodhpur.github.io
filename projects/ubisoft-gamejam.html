Ubisoft Gamejam

Or a lesson for using game engines

13 March 2017

About 20 students were selected to visit Ubisoft Pune office for a 24 hour gamejam. The theme was water, and we came up with a platformer game.

I wrote the game engine in JavaScript from scratch and the artwork is originally created by Bidish Ghosh but this left us with little time to work on the game dynamics. There is not more to describe here but you can play the game here; lightweight and runs in almost any browser.

Thanks to Sarthak Khandelwal, Bidish Ghosh and Yashwardhan Didwania for working with me on this project.
